# Упражнения — Бази данни

## 10.01 — Създаване на база и таблица

1. Отворете `localhost/phpmyadmin`
2. Създайте база `myfirstdatabase`
3. Създайте таблица `users` с колони:
   - `id` — INT, AUTO_INCREMENT, PRIMARY KEY
   - `username` — VARCHAR(30), NOT NULL
   - `pwd` — VARCHAR(255), NOT NULL
   - `email` — VARCHAR(100), NOT NULL
   - `created_at` — DATETIME, DEFAULT CURRENT_TIMESTAMP
4. Добавете 2–3 записа чрез SQL INSERT

---

## 10.02 — SELECT заявки

Използвайки таблицата от 10.01, напишете SQL заявки за:

1. Всички потребители
2. Само потребителите с имейл от `@abv.bg`
3. Потребителите, сортирани по дата на създаване (от най-нов)
4. Броят на всички потребители

---

## 10.03 — PHP + PDO: Регистрация

Създайте:
1. `index.php` — HTML форма за регистрация (username, email, password)
2. `includes/db.php` — PDO връзка с базата
3. `includes/register.php` — записва потребителя в БД

> ⚠️ Хеширайте паролата с `password_hash()` преди записване!

---

## 10.04 — PHP + PDO: Извличане на потребители

Създайте страница, която:
1. Свързва се с базата чрез PDO
2. Извлича всички потребители с `SELECT`
3. Показва ги в HTML таблица

---

## 10.05 — Логин с PDO

Създайте:
1. `login.html` — форма за вход (email + password)
2. `includes/login_user.php` — проверява имейл в БД, верифицира паролата с `password_verify()`, стартира сесия
3. При успешен вход — пренасочване към `homepage.html`
4. При грешка — показване на съобщение

---

## 10.06 — CRUD: Управление на продукти

Създайте таблица `products` (id, name, price, quantity) и PHP страница с:
1. **Форма** за добавяне на нов продукт (INSERT)
2. **Таблица** с всички продукти (SELECT)
3. **Бутон „Изтрий"** до всеки продукт (DELETE)
4. **Бутон „Редактирай"** — зарежда данните във формата за промяна (UPDATE)
