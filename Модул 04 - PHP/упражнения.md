# Упражнения — PHP

## Тема 01 — Основи

### 09.01 — Hello World

Създайте PHP файл, който извежда "Hello World" в HTML страница. Стартирайте го през XAMPP.

### 09.02 — Калкулатор

Създайте HTML форма с две числови полета и падащ списък за операция (+, -, *, /). При изпращане PHP скрипт изчислява и показва резултата.

### 09.03 — Масив от студенти

Създайте асоциативен масив с 5 студента (име, факултетен номер, среден успех). Изведете ги в HTML таблица с `foreach`.

---

## Тема 02 — Функции

### 09.04 — Функция за валидация

Напишете функция `validateEmail(string $email): bool`, която проверява дали имейлът съдържа `@` и `.`.

### 09.05 — Шаблон за страница

Създайте `header.php` и `footer.php` и ги включете в три различни страници с `include`.

---

## Тема 03 — ООП

### 09.06 — Клас Product

Създайте клас `Product` с:
- Свойства: `name`, `price`, `quantity` (private)
- Конструктор
- Getter методи
- Метод `getTotalPrice()` → `price * quantity`

### 09.07 — Клас User с хеширане

Създайте клас `User` с:
- Свойства: `id`, `email`, `name`
- Метод `setPassword($password)` — хешира паролата с `password_hash()`
- Статичен метод `fromArray(array $data): User`

---

## Тема 04 — Web PHP

### 09.08 — Форма за регистрация

Създайте пълна форма за регистрация (username, email, password). PHP скриптът:
1. Проверява дали всички полета са попълнени
2. Проверява дали паролата е поне 6 символа
3. Показва съобщение за успех или грешка

### 09.09 — Login/Logout

Създайте:
1. Форма за логин
2. PHP скрипт, който записва потребителя в `$_SESSION`
3. Страница, достъпна само за логнати потребители
4. Бутон за логаут, който унищожава сесията

### 09.10 — REST-подобен endpoint

Създайте PHP файл, който работи като API:
- **GET** — връща JSON `{"status": "running"}`
- **POST** — приема JSON с `name` и връща `{"greeting": "Hello, {name}!"}`
