# HTML формуляри

## Елементът `<form>`

Формулярите позволяват на потребителите да въвеждат данни и да ги изпращат към сървъра.

```html
<form action="process.php" method="POST">
    <!-- полета тук -->
    <input type="submit" value="Изпрати">
</form>
```

| Атрибут | Описание |
|---------|----------|
| `action` | URL, към който се изпращат данните |
| `method` | HTTP метод — `GET` или `POST` |

### GET срещу POST

| GET | POST |
|-----|------|
| Данните се виждат в URL-а | Данните се изпращат в тялото на заявката |
| Подходящ за търсене | Подходящ за регистрация, логин |
| Ограничен размер на данни | Без ограничение |
| Не е сигурен за пароли | По-сигурен за чувствителни данни |

## Типове `<input>`

```html
<input type="text" name="username" placeholder="Потребителско име">
<input type="email" name="email" placeholder="Имейл" required>
<input type="password" name="password" placeholder="Парола">
<input type="number" name="age" min="1" max="120">
<input type="color" name="favoriteColor">
<input type="date" name="birthday">
<input type="file" name="avatar">
```

| Тип | Описание |
|-----|----------|
| `text` | Обикновено текстово поле |
| `email` | Текстово поле с валидация за имейл |
| `password` | Скрива въведените символи |
| `number` | Числово поле с `min`/`max` |
| `color` | Избор на цвят |
| `date` | Календар за дата |
| `file` | Качване на файл |
| `hidden` | Скрито поле (не се вижда от потребителя) |

## Етикети (`<label>`)

Всяко поле трябва да има етикет за достъпност:

```html
<label for="email">Имейл:</label>
<input type="email" id="email" name="email">
```

Атрибутът `for` свързва етикета с полето чрез `id`. При клик на етикета, полето получава фокус.

## Радио бутони и чекбоксове

**Радио бутони** — само един избор от група:

```html
<p>Възраст:</p>
<label><input type="radio" name="age" value="young"> под 18</label>
<label><input type="radio" name="age" value="adult"> 19-35</label>
<label><input type="radio" name="age" value="senior"> 36+</label>
```

**Чекбоксове** — множествен избор:

```html
<p>Интереси:</p>
<label><input type="checkbox" name="interests" value="html"> HTML</label>
<label><input type="checkbox" name="interests" value="css"> CSS</label>
<label><input type="checkbox" name="interests" value="js"> JavaScript</label>
```

> При радио бутони всички опции имат **едно и също `name`**.

## Падащ списък (`<select>`)

```html
<label for="country">Страна:</label>
<select id="country" name="country">
    <option value="">-- Изберете --</option>
    <option value="bg">България</option>
    <option value="de">Германия</option>
    <option value="uk">Великобритания</option>
</select>
```

Множествен избор:

```html
<select name="skills" multiple size="4">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="js">JavaScript</option>
    <option value="php">PHP</option>
</select>
```

## Текстова област (`<textarea>`)

```html
<label for="comments">Коментари:</label>
<textarea id="comments" name="comments" rows="5" cols="40"
          placeholder="Вашите коментари тук..."></textarea>
```

## Бутони

```html
<!-- Изпращане на формата -->
<input type="submit" value="Изпрати">

<!-- Или с <button> елемент -->
<button type="submit">Изпрати</button>

<!-- Нулиране на формата -->
<input type="reset" value="Изчисти">

<!-- Обикновен бутон (за JS) -->
<button type="button" id="myBtn">Натисни ме</button>
```

## Атрибути за валидация

HTML5 предоставя вградена валидация:

```html
<input type="email" name="email" required>
<input type="text" name="username" minlength="3" maxlength="20" required>
<input type="number" name="age" min="18" max="99">
<input type="text" name="code" pattern="[A-Za-z]{3}[0-9]{3}" title="3 букви и 3 цифри">
```

| Атрибут | Описание |
|---------|----------|
| `required` | Полето е задължително |
| `minlength` / `maxlength` | Минимална / максимална дължина |
| `min` / `max` | Минимална / максимална стойност (за числа) |
| `pattern` | Регулярен израз за валидация |
| `placeholder` | Текст-подсказка в празното поле |

## Пълен пример

```html
<form action="register.php" method="POST">
    <label for="name">Име:</label>
    <input type="text" id="name" name="name" required>

    <label for="email">Имейл:</label>
    <input type="email" id="email" name="email" required>

    <label for="password">Парола:</label>
    <input type="password" id="password" name="password" minlength="6" required>

    <p>Възраст:</p>
    <label><input type="radio" name="ageGroup" value="young"> под 18</label>
    <label><input type="radio" name="ageGroup" value="adult"> 19-35</label>
    <label><input type="radio" name="ageGroup" value="mature"> 36-48</label>
    <label><input type="radio" name="ageGroup" value="senior"> над 49</label>

    <label for="country">Страна:</label>
    <select id="country" name="country">
        <option value="bg">България</option>
        <option value="de">Германия</option>
        <option value="fr">Франция</option>
    </select>

    <label for="bio">За вас:</label>
    <textarea id="bio" name="bio" rows="4"></textarea>

    <button type="submit">Регистрация</button>
</form>
```
