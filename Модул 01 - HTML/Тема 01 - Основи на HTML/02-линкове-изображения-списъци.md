# Линкове, изображения и списъци

## Линкове (Hyperlinks)

Елементът `<a>` създава хипервръзка:

```html
<a href="https://www.fmi.uni-sofia.bg/">ФМИ — Софийски университет</a>
```

### Отваряне в нов таб

```html
<a href="https://www.fmi.uni-sofia.bg/" target="_blank" rel="noopener noreferrer">
    Отвори в нов таб
</a>
```

| Атрибут | Описание |
|---------|----------|
| `href` | URL адресът на дестинацията |
| `target="_blank"` | Отваря линка в нов таб |
| `rel="noopener noreferrer"` | Сигурност при `target="_blank"` |

### Вътрешни линкове

```html
<!-- Линк към друга страница в същата директория -->
<a href="contact.html">Контакти</a>

<!-- Линк към елемент на същата страница -->
<a href="#section2">Отиди към секция 2</a>

<h2 id="section2">Секция 2</h2>
```

## Изображения

Елементът `<img>` е самозатварящ се (няма затварящ таг):

```html
<!-- Изображение от интернет -->
<img src="https://picsum.photos/400/300" alt="Случайна снимка">

<!-- Изображение от същата директория -->
<img src="photo.jpg" alt="Моята снимка" width="300">

<!-- Изображение с грешен път — показва alt текста -->
<img src="missing.jpg" alt="Тази снимка липсва">
```

| Атрибут | Описание |
|---------|----------|
| `src` | Път или URL до изображението |
| `alt` | Алтернативен текст (показва се при липса на снимката) |
| `width` / `height` | Размер в пиксели |

> **Важно:** Атрибутът `alt` е задължителен за достъпност (accessibility). Екранните четци го използват, за да опишат снимката.

## Списъци

### Неномериран списък (Unordered List)

```html
<ul>
    <li>Елемент 1</li>
    <li>Елемент 2</li>
    <li>Елемент 3</li>
</ul>
```

Промяна на маркера чрез CSS:

```html
<ul style="list-style-type: square;">
    <li>Квадратен маркер</li>
    <li>Квадратен маркер</li>
</ul>
```

Стойности за `list-style-type`: `disc` (по подразбиране), `circle`, `square`, `none`.

### Номериран списък (Ordered List)

```html
<!-- Стандартен -->
<ol>
    <li>Първи</li>
    <li>Втори</li>
</ol>

<!-- Започва от 30 -->
<ol start="30">
    <li>Тридесет</li>
    <li>Тридесет и едно</li>
</ol>

<!-- Римски цифри -->
<ol type="I">
    <li>Едно</li>
    <li>Две</li>
</ol>
```

Стойности за `type`: `1` (числа), `A` (главни букви), `a` (малки букви), `I` (римски), `i` (малки римски).

### Вложени списъци

```html
<ul>
    <li>Frontend
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
    </li>
    <li>Backend
        <ul>
            <li>PHP</li>
            <li>MySQL</li>
        </ul>
    </li>
</ul>
```

## Таблици

```html
<table border="1">
    <thead>
        <tr>
            <th>Име</th>
            <th>Години</th>
            <th>Град</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Иван</td>
            <td>22</td>
            <td>София</td>
        </tr>
        <tr>
            <td>Мария</td>
            <td>20</td>
            <td>Пловдив</td>
        </tr>
    </tbody>
</table>
```

| Елемент | Описание |
|---------|----------|
| `<table>` | Контейнер на таблицата |
| `<thead>` | Заглавен ред |
| `<tbody>` | Тяло на таблицата |
| `<tr>` | Ред |
| `<th>` | Заглавна клетка (удебелена) |
| `<td>` | Обикновена клетка |
