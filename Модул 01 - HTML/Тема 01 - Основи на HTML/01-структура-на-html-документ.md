# Структура на HTML документ

## Какво е HTML?

**HTML** (HyperText Markup Language) е стандартният език за маркиране, използван за създаване на уеб страници. Той описва **структурата** на съдържанието — заглавия, параграфи, изображения, линкове и др.

> HTML **не** е език за програмиране — той е език за маркиране (markup language).

## Основна структура

Всеки HTML документ следва тази базова структура:

```html
<!DOCTYPE html>
<html lang="bg">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заглавие на страницата</title>
</head>
<body>
    <!-- Съдържанието на страницата -->
    <h1>Здравей, свят!</h1>
    <p>Това е първият ми HTML документ.</p>
</body>
</html>
```

### Обяснение на елементите

| Елемент | Описание |
|---------|----------|
| `<!DOCTYPE html>` | Декларация за версията на HTML (HTML5) |
| `<html lang="bg">` | Кореновият елемент; `lang` указва езика |
| `<head>` | Метаданни — заглавие, кодиране, стилове |
| `<meta charset="UTF-8">` | Поддръжка на кирилица и специални символи (€, ©) |
| `<meta name="viewport" ...>` | Адаптивност за мобилни устройства |
| `<title>` | Заглавието в таба на браузъра |
| `<body>` | Видимото съдържание на страницата |

## Заглавия (Headings)

HTML предоставя шест нива на заглавия — от `<h1>` (най-важно) до `<h6>` (най-малко важно):

```html
<h1>Основно заглавие</h1>
<h2>Подзаглавие</h2>
<h3>Секция</h3>
```

> **Добра практика:** Използвайте само едно `<h1>` на страница. Заглавията трябва да следват йерархия — не прескачайте нива.

## Параграфи и текст

```html
<p>Това е параграф с обикновен текст.</p>
<p>Това е <strong>важен удебелен</strong> текст.</p>
<p>Това е <em>наклонен</em> текст.</p>
<p>Това е <mark>маркиран</mark> текст.</p>
```

| Елемент | Визуален ефект | Семантично значение |
|---------|---------------|---------------------|
| `<strong>` | Удебелен | Важен текст |
| `<em>` | Наклонен | Акцент |
| `<mark>` | Маркиран (жълт фон) | Подчертана релевантност |
| `<b>` | Удебелен | Без семантика |
| `<i>` | Наклонен | Без семантика |

## Хоризонтална линия и коментари

```html
<h2>Заглавие</h2>
<hr>
<p>Текст след разделителна линия.</p>

<!-- Това е коментар — не се вижда в браузъра -->
```

## Стилов атрибут (inline стил)

Атрибутът `style` позволява директно добавяне на CSS:

```html
<h1 style="color: red;">Червено заглавие</h1>
<p style="color: red;">Целият параграф е <span style="color: blue;">червен, освен тази част</span>.</p>
```

> **Забележка:** Inline стиловете са удобни за бърз тест, но за истински проекти се предпочита външен CSS файл.
