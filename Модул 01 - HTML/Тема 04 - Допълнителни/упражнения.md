# Упражнения — Допълнителни HTML теми

## 04.01 — HTML Entities

Създайте HTML страница, която показва следния текст **буквално** в браузъра (използвайте HTML entities):

```
Тагът <p> създава параграф.
Символът & се използва за entities.
Формулата е: 5 < 10 & 10 > 5.
Между   тези   думи   има   допълнителни   интервали.
```

> **Подсказка:** Използвайте `&lt;`, `&gt;`, `&amp;` и `&nbsp;`.

## 04.02 — Списък с дефиниции

Създайте речник (glossary) на основните HTML термини, използвайки `<dl>`, `<dt>` и `<dd>`. Включете поне 5 термина, например:

- HTML
- CSS
- Tag (Таг)
- Attribute (Атрибут)
- Element (Елемент)

## 04.03 — Текстово форматиране

Създайте HTML страница, която демонстрира:

1. Текст с **изтрита** стара цена и **добавена** нова цена (използвайте `<del>` и `<ins>`)
2. Химическата формула на водата: H₂O (използвайте `<sub>`)
3. Формулата на Айнщайн: E = mc² (използвайте `<sup>`)
4. Теоремата на Питагор: a² + b² = c²

## 04.04 — Свързване на CSS и JavaScript

Създайте проект от три файла:

1. **index.html** — HTML документ, който свързва външен CSS файл и външен JS файл
2. **style.css** — съдържа стилове (например: `body { font-family: Arial; }` и `h1 { color: blue; }`)
3. **script.js** — съдържа `alert("Страницата е заредена!");`

Уверете се, че:
- CSS файлът е свързан с `<link>` в `<head>`
- JS файлът е свързан с `<script>` в края на `<body>`

## 04.06 — Видео и аудио

Създайте HTML страница с:

1. `<video>` елемент с атрибути `controls` и `muted`. Добавете `poster` изображение.
2. `<audio>` елемент с `controls`.
3. Добавете текст, който се показва, ако браузърът не поддържа елемента.

## 04.07 — Canvas и SVG

Създайте HTML страница с две секции:

**Секция 1 — Canvas:**
- Добавете `<canvas>` елемент с `width="400"` и `height="300"`
- С JavaScript нарисувайте: правоъгълник, кръг и линия

**Секция 2 — SVG:**
- Добавете `<svg>` елемент
- Нарисувайте: кръг (`<circle>`), правоъгълник (`<rect>`) и текст (`<text>`)

## 04.08 — Optgroup, Fieldset и Legend

Създайте форма за поръчка на кола с:

1. **Fieldset „Лична информация"** с `<legend>`:
   - Име (текстово поле, `autofocus`)
   - Имейл (тип `email`, `required`)
   - Телефон (тип `tel`)

2. **Fieldset „Избор на кола"** с `<legend>`:
   - Марка — `<select>` с `<optgroup>` за различни държави (напр. Немски: BMW, Audi; Японски: Toyota, Honda)
   - Цвят — `<input type="color">`
   - Бюджет — `<input type="range" min="10000" max="100000">`

3. Бутон за изпращане

## 04.10 — Таблица с thead, tbody и tfoot

Създайте таблица за „Разходи за месеца" с:

- `<thead>` — колони: „Категория", „Сума"
- `<tbody>` — поне 5 реда с разходи (напр. Наем, Храна, Транспорт, и т.н.)
- `<tfoot>` — ред с общата сума (използвайте `colspan` ако е нужно)

Добавете `border="1"` за видимост.

## 04.11 — Булеви атрибути

Създайте форма, която демонстрира различни булеви атрибути:

1. Текстово поле с `autofocus` (автоматичен фокус)
2. Поле с `required` (задължително)
3. Поле с `disabled` (деактивирано, със зададена стойност)
4. Поле с `readonly` (само за четене, със стойност)
5. Чекбокс с `checked` (маркиран по подразбиране)
6. `<select>` с една `<option>` маркирана с `selected`
7. `<details>` с `open`

## 04.12 — Комплексна страница (обобщение)

Създайте уеб страница „Научна статия", която включва **всички** допълнителни елементи от тази тема:

1. **Структура:** `<header>`, `<nav>`, `<main>`, `<footer>`
2. **Заглавие** с формула, използваща `<sup>` (напр. „Теорията E=mc²")
3. **Параграф** с HTML entities (`&lt;`, `&gt;`, `&amp;`)
4. **Списък с дефиниции** за ключови термини
5. **Таблица** с `<thead>`, `<tbody>`, `<tfoot>`
6. **Текст** с `<del>`, `<ins>`, `<sub>` и `<sup>`
7. **Video** елемент с `controls`
8. **SVG графика** — проста диаграма
9. **Форма** за коментар с:
   - `<fieldset>` и `<legend>`
   - `<datalist>` за autocomplete
   - `<optgroup>` в `<select>`
   - Поле с `autofocus`
   - Поле с `required`
10. Свързан външен CSS файл (`<link>`) и JS файл (`<script>`)
11. `<iframe>` с вградено видео

> Целта на това упражнение е да се увери студентът, че познава **всички** HTML елементи и атрибути от курса.
