# Цикли, функции и обекти

## Цикли

### while

Изпълнява се, докато условието е вярно:

```js
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
```

### do...while

Изпълнява се **поне веднъж** преди проверка:

```js
let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
```

### for

Най-често използваният цикъл:

```js
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

### for...in

Обхожда **пропъртитата** на обект:

```js
let person = { name: "Ivan", age: 25, city: "Sofia" };
for (let key in person) {
    console.log(`${key}: ${person[key]}`);
}
```

### for...of

Обхожда **елементите** от масиви и стрингове:

```js
let colors = ["red", "green", "blue"];
for (let color of colors) {
    console.log(color);
}
```

### break и continue

```js
for (let i = 0; i < 10; i++) {
    if (i === 5) break;       // прекъсва целия цикъл
    if (i % 2 === 0) continue; // преминава към следващата итерация
    console.log(i);            // 1, 3
}
```

## Масиви

```js
let numbers = [10, 20, 30, 40, 50];

console.log(numbers[0]);      // 10
console.log(numbers.length);  // 5

numbers.push(60);             // добавя в края
numbers.pop();                // премахва от края
numbers.unshift(0);           // добавя в началото
numbers.shift();              // премахва от началото
```

### Обхождане на масив

```js
let arr = [1, 2, 3, 4, 5];

// for цикъл
for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}

// for...of
for (let num of arr) {
    console.log(num);
}

// forEach
arr.forEach(function(num, index) {
    console.log(`${index}: ${num}`);
});
```

## Функции

### Деклариране

```js
function greet() {
    console.log("Hello!");
}

greet(); // извикване
```

### С параметри

```js
function greet(name) {
    console.log("Hello, " + name);
}

greet("Иван"); // "Hello, Иван"
```

### Връщане на стойност

```js
function sum(a, b) {
    return a + b;
}

let result = sum(3, 5); // 8
```

### Параметри по подразбиране

```js
function greet(name = "гост") {
    console.log(`Здравей, ${name}!`);
}

greet();        // "Здравей, гост!"
greet("Мария"); // "Здравей, Мария!"
```

## Анонимни и Arrow функции

### Анонимна функция (function expression)

```js
let show = function() {
    console.log("Hi");
};

show();
```

### Arrow функция (ES6)

```js
let multiply = (a, b) => a * b;
let square = x => x * x;
let sayHi = () => console.log("Hi!");

console.log(multiply(3, 4)); // 12
console.log(square(5));       // 25
```

### Arrow функция с тяло

```js
let describe = (name, age) => {
    let text = `${name} е на ${age} години.`;
    return text;
};
```

> Arrow функциите са кратък запис, но имат и разлика в поведението на `this`.

## Обекти

### Създаване

```js
let user = {
    name: "Ivan",
    age: 30,
    city: "Sofia"
};

// Или:
let user2 = new Object();
user2.name = "Petur";
```

### Достъп до пропъртита

```js
console.log(user.name);       // "Ivan"
console.log(user["age"]);     // 30
```

### Манипулации

```js
user.isAdmin = true;           // добавяне
user.name = "Georgi";          // промяна
delete user.age;                // изтриване
```

### Метод на обект

```js
let calculator = {
    add: function(a, b) {
        return a + b;
    },
    subtract(a, b) {           // съкратен синтаксис
        return a - b;
    }
};

console.log(calculator.add(5, 3));      // 8
console.log(calculator.subtract(5, 3)); // 2
```

### Обхождане

```js
let user = { name: "Ivan", age: 30, city: "Sofia" };

for (let key in user) {
    console.log(`${key}: ${user[key]}`);
}

// Или:
console.log(Object.keys(user));   // ["name", "age", "city"]
console.log(Object.values(user)); // ["Ivan", 30, "Sofia"]
```
