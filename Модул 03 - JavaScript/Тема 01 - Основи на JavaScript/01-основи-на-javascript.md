# Основи на JavaScript

## Въведение

**JavaScript** е динамичен език за програмиране, който позволява уеб страниците да бъдат интерактивни. Може да се използва в **браузъра** и на **сървъра** (Node.js).

### Какво може JavaScript в браузъра?

- Промяна на съдържанието на страницата без презареждане
- Реакция на потребителски действия (клик, скрол, въвеждане)
- Валидиране на формуляри
- Комуникация със сървъра (AJAX / Fetch)

### Ограничения в браузъра

- Не може да чете/пише произволни файлове от файловата система
- Same Origin Policy — ограничен достъп до други домейни
- Не може да достъпва хардуер директно

## Използване на JavaScript в HTML

### Вграден код

```html
<script>
    alert('Hello world!');
</script>
```

### Външен файл (✅ препоръчително)

```html
<script src="script.js"></script>
```

Или с `defer` — зарежда скрипта, но го изпълнява **след** зареждане на целия HTML:

```html
<script src="script.js" defer></script>
```

### Конзола на браузъра

Отворете DevTools (`F12` или `Ctrl+Shift+I`) → таб **Console**.

```js
console.log("Програмата работи!");
console.warn("Предупреждение!");
console.error("Грешка!");
```

## Променливи

Контейнери за съхранение на данни: числа, текстове, булеви стойности, масиви, обекти.

### Деклариране

```js
let name = "Иван";       // може да се промени
const pi = 3.14;          // НЕ може да се промени
var age = 25;              // стар начин — избягвайте
```

| Ключова дума | Може да се промени | Обхват |
|-------------|-------------------|--------|
| `const` | ❌ | Блоков |
| `let` | ✅ | Блоков |
| `var` | ✅ | Функционален |

### Правила за именуване

- Започва с буква, `_` или `$`
- Чувствително към главни/малки букви (`Name` ≠ `name`)
- Не може да е запазена дума (`let`, `if`, `return`)

### Динамично типизиране

JavaScript определя типа автоматично:

```js
let example = 42;           // number
example = "Здравей свят!";   // string — типът се промени
```

### Основни типове

| Тип | Пример | Описание |
|-----|--------|----------|
| `number` | `42`, `3.14` | Числа (цели и дробни) |
| `string` | `"Hello"`, `'Здравей'` | Текст |
| `boolean` | `true`, `false` | Логически стойности |
| `undefined` | `undefined` | Неинициализирана стойност |
| `null` | `null` | Празна/липсваща стойност |
| `object` | `{ name: "Ivan" }` | Обект |
| `array` | `[1, 2, 3]` | Масив (технически обект) |

> **Добри практики:** Използвай `const` по подразбиране. Описателни имена. Минимален обхват.

## Интерактивни функции

### `alert(message)`

Показва модален диалог с бутон OK:

```js
alert("Добре дошъл!");
```

### `prompt(message, default)`

Поле за въвеждане на текст (връща `string` или `null`):

```js
let name = prompt("Как се казваш?", "Иван");
alert("Здравей, " + name + "!");
```

### `confirm(message)`

Връща `true` или `false`:

```js
let result = confirm("Сигурен ли си?");
alert(result ? "Да" : "Не");
```

> **Съвет:** Използвай `console.log()` за дебъг, `alert`/`prompt`/`confirm` — умерено.

## Оператори

### Аритметични

```js
let x = 5 + 3;    // 8
let y = 7 % 2;    // 1 (остатък)
let z = 2 ** 3;   // 8 (степен)
```

| Оператор | Описание |
|----------|----------|
| `+` | Събиране |
| `-` | Изваждане |
| `*` | Умножение |
| `/` | Деление |
| `%` | Остатък (modulo) |
| `**` | Степен |

### Присвояване

```js
let a = 5;
a += 2;   // a = 7
a -= 1;   // a = 6
a *= 3;   // a = 18
a++;      // a = 19
a--;      // a = 18
```

### Сравнителни

```js
5 == "5"   // true  — сравнява СТОЙНОСТТА (с конверсия)
5 === "5"  // false — сравнява стойността И типа
5 != "5"   // false
5 !== "5"  // true
```

> ⚠️ **Винаги** използвайте `===` и `!==` за избягване на неочаквани конверсии.

### Логически

```js
let age = 20, hasTicket = true;
let canEnter = (age > 18 && hasTicket);  // true
let isFree = (age < 12 || age > 65);    // false
let isBlocked = !canEnter;               // false
```

| Оператор | Описание |
|----------|----------|
| `&&` | Логическо И (and) |
| `\|\|` | Логическо ИЛИ (or) |
| `!` | Логическо НЕ (not) |

### Конкатенация на стрингове

```js
console.log('3' + 2);         // "32" (конкатенация!)
console.log(Number('3') + 2); // 5 (аритметика)
```

> **Внимание:** Операторът `+` при стринг и число извършва **конкатенация**, не събиране.

## Условни конструкции

### if / else if / else

```js
let score = 75;

if (score >= 90) {
    console.log("Отличен");
} else if (score >= 70) {
    console.log("Добър");
} else {
    console.log("Незадоволителен");
}
```

### Тернарен оператор

```js
let message = (age >= 18) ? "Пълнолетен" : "Непълнолетен";
```

### switch

```js
let day = "понеделник";

switch (day) {
    case "понеделник":
    case "вторник":
        console.log("Начало на седмицата");
        break;
    case "петък":
        console.log("Почти уикенд!");
        break;
    default:
        console.log("Обикновен ден");
}
```

## Template Literals

```js
let name = "Иван";
let age = 22;

// Стар начин:
console.log("Здравей, " + name + ". Ти си на " + age + " години.");

// Template literal (ES6):
console.log(`Здравей, ${name}. Ти си на ${age} години.`);
```

> Използвайте обратни кавички (`` ` ``) за template literals.
