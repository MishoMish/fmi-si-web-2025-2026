# Упражнения — Bonus

## 06.01 — Мобилна навигация

Създайте уебсайт с навигация, която:
- На **мобилни** (до 600px): Навигационните връзки са подредени **вертикално**, един под друг
- На **таблети и нагоре** (от 601px): Навигационните връзки са подредени **хоризонтално**, един до друг

```html
<nav class="navbar">
    <a href="#">Начало</a>
    <a href="#">Услуги</a>
    <a href="#">За нас</a>
    <a href="#">Контакти</a>
</nav>
```

```css
/* Мобилна версия */
.navbar {
    display: flex;
    flex-direction: column;
    gap: 10px;
    padding: 10px;
    background: #333;
}

.navbar a {
    color: white;
    text-decoration: none;
    padding: 10px;
}

/* За таблети и нагоре */
@media (min-width: 601px) {
    .navbar {
        flex-direction: row;
        justify-content: center;
        gap: 20px;
    }
}
```

## 06.02 — Responsive Grid за снимки

Създайте галерия от изображения, която:
- На **мобилни**: 1 колона
- На **таблети**: 2 колони
- На **десктопи**: 3 колони

```html
<div class="gallery">
    <img src="image1.jpg" alt="Снимка 1">
    <img src="image2.jpg" alt="Снимка 2">
    <img src="image3.jpg" alt="Снимка 3">
    <img src="image4.jpg" alt="Снимка 4">
    <img src="image5.jpg" alt="Снимка 5">
    <img src="image6.jpg" alt="Снимка 6">

```

```css
.gallery {
    display: grid;
    grid-template-columns: 1fr;
    gap: 15px;
    padding: 20px;
}

.gallery img {
    width: 100%;
    border-radius: 8px;
}

@media (min-width: 601px) {
    .gallery {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (min-width: 1025px) {
    .gallery {
        grid-template-columns: repeat(3, 1fr);
    }
}
```

    background: none;
    border: none;
    color: white;
    font-size: 24px;
    cursor: pointer;
    display: block;
}

.menu {
    display: none;
    list-style: none;

## 06.10 — Комплексен Responsive Layout

Създайте пълна страница, която включва:
- Header с навигация (responsive)
- Main content с 3 картички (1 колона на мобилни, 3 на десктопи)
- Sidebar (под содържанието на мобилни, вдясно на десктопи)
- Footer

```html
<!DOCTYPE html>
<html lang="bg">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Layout</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Мой Сайт</h1>
        <nav>
            <a href="#">Начало</a>
            <a href="#">За нас</a>
            <a href="#">Контакти</a>
        </nav>
    </header>

    <main class="container">
        <section class="content">
            <div class="card">
                <h2>Картичка 1</h2>
                <p>Описание...</p>
            </div>
            <div class="card">
                <h2>Картичка 2</h2>
                <p>Описание...</p>
            </div>
            <div class="card">
                <h2>Картичка 3</h2>
                <p>Описание...</p>
            </div>
        </section>

        <aside class="sidebar">
            <h3>Информация</h3>
            <p>Текст на sidebar...</p>
        </aside>
    </main>

    <footer>
        <p>&copy; 2025 Мой Сайт</p>
    </footer>
</body>
</html>
```

Напишете CSS, което:
- На мобилни: sidebar е под содържанието
- На таблети/десктопи: sidebar е вдясно, картичките са в 3 колони на десктопи
- Адаптивен шрифт на всички нива
