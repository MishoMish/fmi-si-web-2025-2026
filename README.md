# Уеб програмиране — СИ 2025/2026

Курс по уеб програмиране за специалност **Софтуерно Инженество**, ФМИ, СУ „Св. Климент Охридски".

## Структура на курса

| Модул | Теми | Описание |
|-------|------|----------|
| **01 — HTML** | Основи, семантични елементи, формуляри | Изграждане на структурата на уеб страници |
| **02 — CSS** | Основи, Box Model, Flexbox | Визуално оформление и подредба |
| **03 — JavaScript** | Променливи, цикли, функции, DOM | Интерактивност на клиентската страна |
| **04 — PHP** | Синтаксис, ООП, Web PHP | Сървърно програмиране |
| **05 — Бази данни** | RDBMS, MySQL, PDO | Съхранение и извличане на данни |
| **06 — Уеб приложение** | Fetch API, автентикация | Свързване на frontend и backend |

## Организация на файловете

```
fmi-si-web-2025-2026/
├── Модул 01 - HTML/
│   ├── Тема 01 - Основи на HTML/
│   ├── Тема 02 - HTML5 семантични елементи/
│   └── Тема 03 - Формуляри/
├── Модул 02 - CSS/
│   ├── Тема 01 - Основи на CSS/
│   ├── Тема 02 - Box Model/
│   └── Тема 03 - Flexbox/
├── Модул 03 - JavaScript/
│   ├── Тема 01 - Основи на JavaScript/
│   ├── Тема 02 - Цикли, функции и обекти/
│   └── Тема 03 - DOM манипулация/
├── Модул 04 - PHP/
│   ├── Тема 01 - Въведение в PHP/
│   ├── Тема 02 - Функции и файлове/
│   ├── Тема 03 - ООП в PHP/
│   └── Тема 04 - Web PHP/
├── Модул 05 - Бази данни/
│   ├── Тема 01 - Релационни бази данни/
│   ├── Тема 02 - MySQL/
│   └── Тема 03 - PDO/
└── Модул 06 - Уеб приложение/
    ├── Тема 01 - Frontend-Backend комуникация/
```

Всяка тема съдържа уроци (`.md` файлове) и файл с упражнения.

## Необходим софтуер

- Текстов редактор — [VS Code](https://code.visualstudio.com/)
- Браузър — Chrome / Firefox (с DevTools)
- Локален сървър — [XAMPP](https://www.apachefriends.org/) (Apache + MySQL + PHP)
